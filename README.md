# Languages benchmark

## Environment

- `node`(required)
- `gcc`
- `go`
- `javac`
- `java`
- `python3`

缺少工具会缺失相关语言数据

## Quick start

```shell
npx @luooooob/languages-benchmark
```

## Development

```shell
git clone https://github.com/luooooob/languages-benchmark.git
cd languages-benchmark
npm run test
```
